# scala-cli-maven-failsafe-spring-pmd-mockito-cucumber-testng-car-data

## Description
A POC for maven app using testng
mockito, and cucumber framework
 with pmd
and failsafe plugin.

## Tech stack
- scala
- maven
	- spring
  - testng
  - failsafe
  - pmd
  - cucumber
	- mockito


## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
